+ ## 개임소개

+ ## 게임설명  
> ### **짜증나는 모기를 잡으며 강해지자**  
>  모기한테서 오래 살아남을 수록 **고혈압 게이지 상승**  
>>  고혈압 게이지로 "Skill" 사용 가능
>  모기를 잡을 수록 **MosqCoin Get**  
>>  MosqCoin으로 아이템 구매 가능
